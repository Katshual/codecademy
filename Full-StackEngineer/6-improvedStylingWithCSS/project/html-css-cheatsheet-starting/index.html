Bases de HTML et CSS

1/ Introduction au développement Web

Le développement web repose sur 2 piliers fondamentaux :

  1. HTML: Langage de balisage utilisé pour structurer le contenu d'une page web,
  2. CSS: Langage de style utilisé pour définir l'apparence de ce contenu.

2/ Découverte des balises HTML

Les balises HTML structurent une page et sont toujours entourées de chevrons.

Chaque balise a un rôle précis.

Balise Usage Exemple
<html> Racine du Document <html>...</html>
<head> Métadonnées <head>...</head>
<body> Contenu visible <body>...</body>
<p> Paragraphe <p>Texte</p>
<h1> à <h6> Titres de niveau 1 à 6 <h1>Titre Principal</h1>
<a> Lien hypertexte <a href="url">Texte du lien</a>
<img> Image <img src="image.jpg" alt="Texte alternatif">

3/ Structurer le texte avec des balises

Balise Usage Exemple
<strong> Texte en gras <strong>Important</strong>
<em> texte en italique <em>Accentué</em>
<ul> Liste non ordonnée <ul><li>Element</li><ul>
<ol> Liste ordonnée <ol><li>Element</li></ol>
<Table> Tableau <table>Cellule</table>
